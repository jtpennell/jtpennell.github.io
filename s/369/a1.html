SVD Based Data Compression¶

Below is code which performs a singular value decomposition of an image, in this we use a greyscale picture of a baboon's face in pgm format. As a result it illustrates the image as $A = UDV^T$, where $D$ is a diagonal matrix with nonnegative entries, and both $U$ and $V$ are orthogonal.

from matplotlib import pyplot
import re
import numpy


def read_pgm(filename, byteorder='>'):
    """Return image data from a raw PGM file as numpy array.

    Format specification: http://netpbm.sourceforge.net/doc/pgm.html

    """
    with open(filename, 'rb') as f:
        buffer = f.read()
    try:
        header, width, height, maxval = re.search(
            b"(^P5\s(?:\s*#.*[\r\n])*"
            b"(\d+)\s(?:\s*#.*[\r\n])*"
            b"(\d+)\s(?:\s*#.*[\r\n])*"
            b"(\d+)\s(?:\s*#.*[\r\n]\s)*)", buffer).groups()
    except AttributeError:
        raise ValueError("Not a raw PGM file: '%s'" % filename)
    return numpy.frombuffer(buffer,
                            dtype='u1' if int(maxval) < 256 else byteorder+'u2',
                            count=int(width)*int(height),
                            offset=len(header)
                            ).reshape((int(height), int(width)))


def display_pgm(array,title): #displays the matrix as a pgm
    pyplot.imshow(array, pyplot.cm.gray)
    pyplot.axis('off')
    pyplot.title(title)
    pyplot.show()


image = read_pgm("baboon.pgm")

#Problem 2: a) Perform an SVD of my facial image and illustrate the results
print("An SVD of the facial image illstrated below, where A = UDV^T.")

U,s,V = numpy.linalg.svd(image)
display_pgm(image,"A")

#Mapping the values of the matrix A = UDV^T linearly to the range [0,255] and then displaying the image
def map_display(array,title): 
    a_min = array.min()
    a_max = array.max()
    
    for i in range(len(array)):
        for j in range(len(array[i])):
            array[i][j] = ((255 * array[i][j] - a_min)/(a_max - a_min))
    
    display_pgm(array,title)

D = numpy.diag(s)
V = numpy.transpose(V)

map_display(U,"U")
map_display(D,"D")
map_display(V,"V^T")

An SVD of my facial image illstrated below, where A = UDV^T.

In the SVD above, where $A = UDV^T$, we see that the matrix U shows our left singular vectors of A. Matrix D is the diagonal values of the first $r$ entries, where $r$ is the rank of A in descending order greater than zero. Finally we see matrix V, which are our right singular vectors of A.

def approximate_A(U,s,V):
    p = [1,2,3,4,5,10,20,30,40,90] #p singular values of the compressed image
    for x in p: # for each value in the list p, approximate A
        A_p = numpy.matrix(U[:, :x]) * numpy.diag(s[:x]) * numpy.matrix(V[:x, :]) #slice row and column values in matrices UDV^T
        
        for i in range(A_p.shape[0]): #for each row, and column value in the approximated matrix, scale
            
            for j in range(A_p.shape[1]):
                
                #if value in the matrix is less than 0 set to 0 or greater than 225 set to 225 
                if A_p[i,j] < 0:
                    A_p[i,j] = 0
                elif A_p[i,j] > 255:
                    A_p[i,j] = 255
                
        display_pgm(A_p,x)

    return

approximate_A(U,s,V)
